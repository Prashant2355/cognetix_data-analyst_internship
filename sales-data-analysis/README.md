# Sales data analysis using Python and Jupyter Notebook.

This project focuses on performing **Exploratory Data Analysis (EDA)** on a sales dataset to uncover business insights, revenue trends, and performance drivers using **Python** and data visualization techniques.

## Objective :-
- To understand overall sales performance
- To identify top-performing product lines and regions
- To analyze order status impact on revenue
- To discover seasonal and yearly sales trends
- To support data-driven business decision-making

## Dataset Overview :-
- **Dataset Name**: Sales Data Sample
- **Records**: Orders across multiple years
- **Key Columns**:
   - Product Line
   - Sales
   - Order Status
   - Country & State
   - Month & Year
   - Customer & Order Details
